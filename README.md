## Fasta to FastQ

Converts Fasta files to FastQ format. 

Example running fasta2fastq.pl using sample.fasta file provided.

Please input the name of a file to be read.
```
sample.fasta
```

Please input the name of the file to print to.
```
temp.fastq
```
